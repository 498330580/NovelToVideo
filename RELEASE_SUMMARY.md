3. **文档** - README、CHANGELOG、RELEASE_NOTES

### 部署方式

#### 方式1：本地运行
```bash
# 激活虚拟环境
.venv\Scripts\Activate.ps1

# 安装依赖
pip install -r requirements.txt

# 启动应用
python run.py

# 访问：http://127.0.0.1:5000
```

#### 方式2：Docker部署（推荐）
```bash
# 启动容器
docker-compose up -d

# 访问：http://localhost:5000
```
